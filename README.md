# MTT实验全能助手

一个专为MTT实验设计的Python计算器，帮助您快速完成细胞计数、单药梯度配制和双药混合配制。

## 功能特点

### 1. 细胞计数与铺板计算
- 支持稀释倍数计算（5倍、10倍等）
- 自动计算原液细胞密度
- 计算铺板所需细胞悬液和培养基体积
- 支持多块96孔板计算

### 2. 单药梯度配制
- 支持多种浓度单位（μM、nM、mM）
- 自动计算每个浓度梯度所需的母液和培养基体积
- 适合Row Drug或Col Drug的梯度设置

### 3. 双药混合配制
- 专门针对Row Drug A + Row Drug B的混合配制
- 支持批量输入浓度组合
- 自动计算混合液中两种药物的体积配比

## 使用方法

### 运行程序
```bash
python mtt_assistant.py
```

### Tab 1: 细胞计数与铺板

**输入参数：**
- 计数板总细胞数：您计数得到的细胞总数
- 计数的格数：通常为4个大格
- 计数前稀释倍数：如果稀释了5倍或10倍，选择对应倍数
- 目标每孔细胞数：如5000个/孔
- 每孔体积：通常0.09 mL（默认值，可自行调整）
- 每块实际使用孔数：如常规仅用 72 孔，可在此设置
- 计划铺板数量：需要铺几块板（默认 96 孔板，可部分使用）
- 配液余量：建议留2-3 mL余量

**输出结果：**
- 原液细胞密度
- 铺板液目标密度
- 需要取多少细胞悬液和培养基

### Tab 2: 单药梯度配制

**输入参数：**
- 药物母液浓度：单位mM
- 目标浓度单位：选择μM、nM或mM
- 每种浓度配制体积：建议1000 μL（足够复孔）
- 浓度梯度：用逗号分隔，如 `0, 5, 10, 20, 50, 100`

**输出结果：**
- 每个浓度梯度需要取的母液体积（μL）
- 需要加的培养基体积（μL）

### Tab 3: 双药混合配制

**输入参数：**
- 药A和药B的母液浓度（mM）
- 每管配制体积（μL）
- 目标浓度单位
- 混合梯度清单：每行一个组合，格式为 `A浓度, B浓度`

**示例输入：**
```
10, 0
10, 10
10, 20
20, 20
```

**输出结果：**
- 每个组合需要加的药A体积（μL）
- 每个组合需要加的药B体积（μL）
- 需要加的培养基体积（μL）

## 实验建议

1. **边缘效应**：96孔板最外圈（Row A/H, Col 1/12）建议不种细胞或只加PBS封边，避免蒸发影响结果。

2. **复孔设置**：建议n=3到n=5，剩余孔可做溶剂对照和空白对照。

3. **DMSO浓度**：如果药物溶解在DMSO中，注意混合后DMSO终浓度不要超过0.1-0.5%（根据细胞类型调整）。

4. **配液余量**：建议多配20-30%的余量，避免操作损耗导致不够用。

## 注意事项

- 所有体积单位：母液浓度用mM，配制体积用μL，铺板体积用mL
- 浓度换算：程序会自动处理单位换算（mM ↔ μM ↔ nM）
- 如果计算结果出现"浓度过高"提示，说明母液浓度不足以配制目标浓度，需要提高母液浓度或降低目标浓度
